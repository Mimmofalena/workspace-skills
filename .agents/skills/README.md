# Project Skills

This project has the following AI skills installed via `workspace-skills`.
You can reference these skills when they are relevant to the current task.

| Skill | Description | Path |
|-------|-------------|------|
| brainstorming | Use this before any creative work - creating features, building components, adding functionality, or modifying behavior. Explores user intent, requirements and design before implementation. | .agents/skills/brainstorming/SKILL.md |
| caveman | Ultra-compressed communication mode. Cuts token usage ~75% by speaking like caveman while keeping full technical accuracy. | .agents/skills/caveman/SKILL.md |
| frontend-design | Create distinctive, production-grade frontend interfaces with high design quality. Use this skill when the user asks to build web components, pages, artifacts, posters, or applications. | .agents/skills/frontend-design/SKILL.md |
| grill-me | Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree. Use when user wants to stress-test a plan, get grilled on their design, or mentions 'grill me'. | .agents/skills/grill-me/SKILL.md |
| ui-ux-pro-max | UI/UX design intelligence for web and mobile. Includes 50+ styles, 161 color palettes, 57 font pairings, 161 product types, 99 UX guidelines, and 25 chart types across 10 stacks. | .agents/skills/ui-ux-pro-max/SKILL.md |
| vercel-react-best-practices | React and Next.js performance optimization guidelines from Vercel Engineering. This skill should be used when writing, reviewing, or refactoring React/Next.js code to ensure optimal performance patterns. | .agents/skills/vercel-react-best-practices/SKILL.md |

## How to use

- **opencode**: skills are automatically available under `.agents/skills/`.
- **GitHub Copilot**: use `.github/prompts/` for modular prompts, or consult `.github/copilot-instructions.md` for global rules.
- **You**: ask the AI to 'use the X skill' when relevant.
